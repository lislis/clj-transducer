Transducers

  • are blueprints
  • of functions that work on sequences
  • that are ignorant of the sequence

Every function that operates on a sequence implements a transducer.

The transducer is the blueprint that is returned if that function is not given a sequence to operate on.

Because the function knows what to do with a sequence (if it had one).
